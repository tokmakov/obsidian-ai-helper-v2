# AI Helper for Obsidian

AI-ассистент прямо в Obsidian с поддержкой любых провайдеров, совместимых с OpenAI API.

## Возможности

- 💬 Чат с AI прямо в боковой панели Obsidian
- 🔄 Поддержка любого провайдера с OpenAI-совместимым API
- 📚 История сессий с автосохранением
- 📋 Копирование текста и блоков кода из ответов
- ⚙️ Гибкие настройки: модель, шрифт, хранение данных
- 📝 Опциональное логирование запросов и ответов

## Установка

### Вручную

1. Скачай последний релиз со страницы (https://github.com/evgeniy/obsidian-ai-helper/releases)
2. Распакуй архив в папку `.obsidian/plugins/obsidian-ai-helper/` внутри твоего vault
3. В Obsidian открой **Настройки → Сторонние плагины** и включи **AI Helper**

### Из исходников

```bash
git clone https://github.com/evgeniy/obsidian-ai-helper
cd obsidian-ai-helper
npm install
npm run build
```

Скопируй `main.js`, `manifest.json` и `styles.css` в `.obsidian/plugins/obsidian-ai-helper/` внутри твоего vault.

## Настройка

Открой **Настройки → AI Helper**:

### Подключение к провайдеру

| Параметр | Описание                | Пример                      |
|----------|-------------------------|-----------------------------|
| Base URL | Адрес API провайдера    | `https://api.openai.com/v1` |
| API Key  | Ключ доступа провайдера | `sk-...`                    |

После ввода Base URL и API Key нажми **Загрузить модели** — плагин автоматически получит список доступных моделей. Выбери нужную из списка.

### Совместимые провайдеры

Плагин работает с любым провайдером, поддерживающим OpenAI-совместимый API:

- https://ai-mediator.ru
- https://routerai.ru/
- https://proxyapi.ru/

### Внешний вид

- **Размер шрифта** — размер текста в панели чата (10–24px)

### Хранение данных

| Параметр       | По умолчанию | Диапазон     |
|----------------|--------------|--------------|
| Хранить логи   | 30 дней      | 10–100 дней  |
| Хранить сессии | 100 дней     | 10–1000 дней |

### Логирование

Включи **Логирование** если хочешь сохранять все запросы и ответы в файлы. Логи хранятся в `.obsidian/plugins/obsidian-ai-helper/logs/` и автоматически удаляются по истечении срока хранения.

## Использование

### Открыть чат

Нажми иконку 🤖 в левой панели Obsidian или используй Command Palette: **AI Helper: Открыть чат с AI**.

### Интерфейс чата

```
┌───────────────────────────────────┐
│ AI Helper   модель   упр.сессиями │  ← шапка
│ [                               ] │  ← список сессий
├───────────────────────────────────┤
│                                   │
│  Сообщения...                     │  ← область чата
│                                   │
├───────────────────────────────────┤
│ [                               ] │  ← поле ввода
│ Ctrl+Enter для отправки           │
└───────────────────────────────────┘
```

### Управление сессиями

| Кнопка | Действие            |
|--------|---------------------|
| ➕     | Создать новый чат   |
| 🗑️     | Удалить текущий чат |
| ✖️     | Закрыть панель      |

Для переключения между сохранёнными сессиями используй выпадающий список в шапке. Название сессии формируется автоматически из первых слов твоего первого вопроса.

### Работа с ответами

- **Выделение текста** — просто выдели мышью и скопируй
- **Копирование кода** — нажми иконку копирования в правом верхнем углу блока кода
- **Вставить в заметку** — кнопка под сообщением ассистента вставляет ответ в текущую открытую заметку

## Хранение данных

Все данные плагина хранятся внутри vault:

```
.obsidian/plugins/obsidian-ai-helper/
├── main.js
├── manifest.json
├── styles.css
├── data.json          # настройки плагина
├── logs/              # логи запросов (если включено)
│   └── 2026-03-26.log
└── sessions/          # история сессий
    └── 2026-03-26T11-47-43-000Z.json
```

При удалении плагина Obsidian удаляет всю папку вместе с логами и сессиями.

Дополнительная информация для разработчиков — в файле `DEVELOPMENT.md`
